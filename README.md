Программа для гимнастики глаз. С определённой переодичностью предлагает пользователю несколько упражнений для отдыха глаз и поддержания зрения в хорошем состоянии при ежедневной работе перед монитором. 
При первом запуске в GUI настраивается периодичность срабатывания и длительность выполнения каждого упражнения. После запуска окно сворачивается в трей.
По завершению периода срабатывания появляется окно с описанием конкретного упражнения и прогресс бааром, отражающим время выполнения.

Программа разработана на dotnet win forms.
